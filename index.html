Правила використання кольорів

  1. Основний колір бренду — коралово-червоний (#FC4E4D), використовуйте його для головних елементів дизайну.
  2. Світлий альтернативний колір (#FD6B6A) застосовуйте для виділення та другорядних елементів.
  3. Темний альтернативний колір (#EC3F3E) використовуйте для акцентування та кнопок дії.
  4. Чорний колір (#000000) обов'язковий для основного тексту та заголовків.
  5. Темно-сірий (#525252) застосовуйте для допоміжного тексту та дрібних елементів.
  6. Світло-сірий (#F8F8F8) використовуйте для фонових блоків та розділювачів.
  7. Логотип завжди використовується в оригінальних кольорах (коралово-червоний з зеленим динозавром).
  8. Дотримуйтесь єдиної кольорової гами на всіх рекламних матеріалах та сторінках сайту.
  9. Елементи розпродажу відображайте виключно червоними мітками на контрастному фоні.
  10. При розробці нових елементів дизайну обов'язково враховуйте співвідношення основного та альтернативних кольорів для збереження цілісності бренду.

Правила використання шрифтів

  1. Заголовки та акценти: Використовуйте Rubik Bold для всіх головних заголовків сторінок та банерів, щоб створити потужний візуальний вплив.
  2. Відеоконтент і соцмережі: Для текстових накладань у відео (Instagram, TikTok, рекламні матеріали) використовуйте виключно Caption (Rubik Bold), вагою 700 для максимальної читабельності.
  3. Ієрархія тексту: Дотримуйтесь чіткої ієрархії від H1 (32px) до H6 (10px) для всього текстового контенту сайту, використовуючи Roboto Slab з відповідною вагою.
  4. Основний текст: Для абзаців та описів товарів завжди використовуйте Roboto Slab 14px, це забезпечить легке читання на всіх пристроях.
  5. Консистентність: Не змішуйте додаткові шрифти окрім визначених Rubik Bold та Roboto Slab у рамках одного проекту чи сторінки.

Примітка: Рекламний логотип призначений для використання у великому розмірі на рекламних матеріалах, банерах та промо-матеріалах. Мінімальна рекомендована ширина - 400px.

Правила використання логотипу

  1. Збереження пропорцій: Забороняється розтягувати, стискати або деформувати логотип будь-яким чином. Співвідношення сторін повинно залишатися постійним незалежно від розміру.
  2. Безпечна зона: Навколо логотипу слід завжди залишати вільний простір, що дорівнює висоті літери "T" в слові "Tea-Rex". Жодні елементи дизайну не повинні потрапляти в цю зону.
  3. Кольорова відповідність: Для світлого фону використовуйте темну версію логотипу, для темного фону — світлу версію. Забороняється змінювати кольорову гаму логотипу або використовувати градієнти.
  4. Цілісність елементів: Забороняється видаляти, додавати або змінювати будь-які елементи логотипу (динозавр, чайник, текстова частина). Логотип повинен використовуватися тільки як єдине ціле.
  5. Мінімальний розмір: Для збереження розпізнаваності логотип не повинен використовуватися в розмірі меншому ніж 20 мм у ширину для друкованих матеріалів та не менше 10% від ширини екрану для цифрових носіїв.
